Key

A number from 1 to 25.

History and Facts

Julius Caesar enciphered messages to his generals by shifting each letter in the message three places farther into the alphabet, so that a would be written as D, b as E, c as F and so on.

If the algorithm is generalized to be shifting letters any number of places, then the key is the number of places to shift. For the English alphabet, that means that keys from 1-25 are valid.

If you choose the key value of 13, you get the ROT13 cipher, which is sometimes used to obfuscate information in internet postings.

Level of Security

The Caesar cipher offers no real security for a couple of reasons. First, if you know that text is encoded with the Caesar cipher, then there are only 25 possible keys, so it is pretty easy to check each one until the message appears. Second, even if you don't know the algorithm, the ciphertext is vulnerable to simple frequency analysis.

Still, the Caesar cipher can be used for "entertainment" encryption, generating cryptograms like those found in many newspapers.
